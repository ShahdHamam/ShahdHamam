### 🌟 Featured Project: AeroCare
**AeroCare** is an IoT-based environmental monitoring system specifically designed to help **asthma patients** track and manage indoor air quality. 
* **How it works:** Integrates an ESP32 microcontroller with a Firebase backend and a Flutter mobile application.
* **Metrics Tracked:** Monitors PM2.5, VOCs, CO2, NO2, temperature, and humidity to detect potential asthma triggers.
* **Key Feature:** Prioritizes immediate user safety by triggering real-time alerts the moment *any individual sensor* crosses its specific safety threshold, ensuring rapid response to hazardous respiratory conditions.
